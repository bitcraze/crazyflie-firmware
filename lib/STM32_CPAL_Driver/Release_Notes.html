STM32 CPAL Library Drivers  update History

V1.2.0 / 21-December-2012

Main Changes

  • Add support for STM32F4xx devices
    • Add new HAL driver cpal_i2c_hal_stm32f4xx.c/.h
    • Update cpal_i2c.c driver to support STM32F4xx devices.
  • Add support for STM32L1xx Ultra Low Power High-Density devices
    • Update cpal_i2c.c driver to support STM32L1xx High-Density devices.
  • Hal Layer
    • Update CPAL_I2C_HAL_DMATXConfig and CPAL_I2C_HAL_DMARXConfig function by setting number of channel used with DMA for STM32F2xx and STM32F4xx devices.
  • Communication Layer
    • Add new option CPAL_OPT_I2C_NACK_ADD.
    • Fix the implementation of CPAL_I2C_RXTC_UserCallback() and CPAL_I2C_TXTC_UserCallback() UserCallback functions.
    • Fix Timeout management mechanism.
    • Add support for no-stop generation mode for master transmitter and slave receiver in DMA and interrupt mode.
    • Add support of listen mode for slave.
    • Add support of null command for master only in interrupt mode.

V1.1.0 / 17-June-2011

Main Changes

  • Add support for STM32F2xx devices.
    • Add new HAL driver cpal_i2c_hal_stm32f2xx.c/.h
    • Update cpal_i2c.c driver to support STM32F2xx devices.
  • cpal_i2c.c
    • Update 2 bytes reception procedure with fix of POS bit control for STM32L1xx and STM32F2xx devices.

V1.0.0 / 14-March-2011

  1. General
  • First Official version supporting I2C peripheral of STM32F10x and STM32L1xx Ultra Low Power Medium-Density devices.
Important Note:
The files cpal_conf_template.h and cpal_usercallback_template.c are mandatory files but are not part of CPAL library drivers. A template of these files are provided in template and examples folders; these files can be modified by user in order to customize and/or configure CPAL library components.
