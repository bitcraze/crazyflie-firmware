# AI-deck State Streamer for Crazyflie 2.x

This folder contains the app layer application for the Crazyflie to stream the state to the NINA module on the AI-deck, which can then be sent over WiFi alongside the camera image.

It is the Crazyflie-side counterpart to the nanocockpit GAP8 streamer in the [aideck-gap8-examples](https://github.com/bitcraze/aideck-gap8-examples) repository. Both sides need to be flashed for the state to reach the host.

See App layer API guide and build instructions [here](https://www.bitcraze.io/documentation/repository/crazyflie-firmware/master/userguides/app_layer/)
